## 📍 HOW TO NAVIGATE THIS PROJECT

### Reading the Theory
1. Open `01_Theory_Documents/00_MASTER_INDEX_Curriculum.md` first
2. Read the Master Index completely (10-15 min)
3. Then read each theory doc in order (one per day, Days 1-7)
4. Reference docs 4-7 during Week 2 for specific topics

### Using the Dashboard
1. Open `03_Interactive_Dashboard/DASHBOARD_Progress_Tracker.html` in your browser
2. Bookmark it
3. Visit daily to track progress
4. Click day number to mark complete
5. Watch completion % increase

### Tracking Your Conversation
1. Keep `04_Chat_Sessions_&_References/CHAT_SESSION_TRANSCRIPT_2026-04-21.pdf` saved
2. This is your record of decisions
3. Reference for understanding why choices were made
4. Share with team if collaborating

### Saving Your Work
1. Create artifacts in Claude.ai
2. Download all artifacts daily
3. Save to `07_Final_Deliverables/`
4. Organize by week
5. By Day 14, you have complete portfolio

### Documenting Learning
1. After each day, create reflection in `05_Progress_Tracking/`
2. What did you learn?
3. What was hard?
4. What blockers came up?
5. What will you focus on tomorrow?
